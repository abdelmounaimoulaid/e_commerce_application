# Shopino - Application E-commerce Flutter

**Shopino** est une petite application mobile **E-commerce** développée avec **Flutter**, permettant d’afficher une liste de produits et de consulter leurs détails.

---

## Fonctionnalités

### 1. Page d’authentification
- Champs : **email** et **mot de passe**
- Bouton **Se connecter** pour accéder à l’application

### 2. Page d’accueil
- Affiche le **nom et le logo** de l’application : **Shopino**
- Bouton menant à la **liste des produits**

### 3. Liste des produits
- Affichage d’une **liste de produits** avec :
  - Nom
  - Prix
  - Image

### 4. Page de détail d’un produit
- Affiche :
  - Image du produit
  - Nom du produit
  - Prix
  - Courte description
- Bouton **Commander** pour passer à la confirmation

### 5. Page de confirmation
- Après avoir cliqué sur **Commander**, affiche un **message de confirmation** à l’utilisateur

---

## Technologies utilisées
- [Flutter](https://flutter.dev/) pour le développement mobile multiplateforme
- [Dart](https://dart.dev/) comme langage de programmation
